## 2) Concise Architecture Diagram

```mermaid
flowchart LR
	A[Flutter App\nAuth + Home + Delivery + Payment UI] --> B[Firebase Auth]
	A --> C[Node.js Express API]
	B --> C

	C --> D[(MongoDB)]
	C --> E[Razorpay API]
	E --> C
	E --> F[/Webhook: /v1/payments/webhook/]
	F --> C

	C --> G[Nominatim Geocode]
	C --> H[OSRM Routing]

	D --> C
	C --> A
```
